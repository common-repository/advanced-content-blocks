== Description ==

**Advanced Content Blocks** – is a set of cool content blocks for the new WordPress editor.

**REQUIRE [GUTENBERG](https://wordpress.org/plugins/gutenberg/) PLUGIN TO BE INSTALLED**

**Currently available blocks:**

* Tabs

**Developing more blocks actively**

Languages:

* English

**This is a new plugin, so please use support forums if you have found any bug or have any other question please use forums to contact us! :)**

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/advanced-content-blocks` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Now plugin is active and you can use new blocks in editor
